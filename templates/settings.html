API 키 설정 안내

API 키는 프로젝트 폴더의 .env 파일에서 설정합니다.

① 네이버 검색 API (무료)
  1. developers.naver.com 접속
  2. Application 등록 → 검색 API 선택
  3. Client ID / Client Secret 발급
  4. .env에 입력
② Claude AI API
  1. console.anthropic.com 접속
  2. API Keys → Create Key
  3. .env에 입력
③ Gmail 이메일 알림
  1. 구글 계정 → 2단계 인증 활성화
  2. 앱 비밀번호 생성 (16자리)
  3. .env에 입력
